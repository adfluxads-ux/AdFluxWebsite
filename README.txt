AdFluxWebsite - Starter Template
----------------------------------

Yeh folder ek ready-to-open starter website hai. Steps:

1) Download aur extract karlo (agar zip use kar rahe ho).
2) Folder ke andar 'index.html' milega.
3) File ko open karne ke liye:
   - Windows: index.html pe right-click -> Open with -> Chrome/Edge
   - Mac: index.html pe double-click ya Open with -> Safari/Chrome
4) Agar background image add karni ho to:
   - assets folder mein 'bg.jpg' rakh do
   - index.html mein <style> ke hero-bg background-image ke end mein url('bg.jpg') add kar do.
5) Text change karna ho to index.html ke andar <h1>, <p> etc. edit karo.
6) Production ke liye:
   - Tailwind CDN development ke liye acha hai. Production mein Next.js + Tailwind build use karo.
   - Host on Vercel / Netlify for free deployment.

Agar chaho to main ye project Vercel pe deploy karke live link bhi bana doon. Batao.
